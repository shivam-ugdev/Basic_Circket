3D Cricket Bowling System (Unity)
A Unity 3D cricket bowling system featuring realistic *swing and spin mechanics*, bounce targeting, and a timing-based control meter.

Features
-  Bounce point targeting (accurate projectile motion)
-  Swing bowling (curves in air, stops after bounce)
-  Spin bowling (turns only at bounce)
-  Bowling meter (0–15 strength based on timing)
-  UI mode switching (Swing / Spin)
-  Bowling cooldown system

 Tech
- Unity (3D)
- C#
- Rigidbody Physics
- Unity UI
